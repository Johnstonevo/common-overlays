Descsription.  

1080P stylized overlay designed for 2-screen arcade emulators.  It uses a 1760*660 centered display window.  Contains a build-in "grid" effect.   

Files

arcade-doublewide.png/cfg - with grid
arcade-doublewide-sample.png - sample screenshot of above

retroarch.cfg settings for ideal use of the overlay 

video_fullscreen_x = 1920
video_fullscreen_y = 1080
custom_viewport_width = 1760
custom_viewport_height = 660
custom_viewport_x = 80
custom_viewport_y = 210
aspect_ratio_index = 22
input_overlay = "/opt/retropie/emulators/retroarch/overlays/arcade-doublewide.cfg"
input_overlay_enable = true
input_overlay_opacity = 1.000000
input_overlay_scale = 1.000000
